# Assistente de Suporte Técnico com IA

Projeto de assistente virtual para suporte técnico, desenvolvido em Python usando a API Groq.

## Como usar

1. Crie um arquivo `.env` com sua chave da Groq:
GROQ_API_KEY=sua_chave_aqui

2. Instale as dependências:

```bash
pip install -r requirements.txt

3. Executar o Projeto
python main.py
